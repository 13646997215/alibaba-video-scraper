快速结论

如果 health 正常 但 scrape 失败，通常是目标站返回了反爬校验页（Captcha/Punish），不是你的前端或 API 路由坏掉。

建议排查顺序

  1. 在“诊断中心”运行全量诊断，查看 diag/scrape/extract 返回。
  2. 若出现 ANTI_BOT_BLOCKED，说明被目标站风控。
  3. 尝试更换商品链接、降低请求频率、稍后再试。
  4. 优先使用“全资源模式”判断是否仍可提取图片等资源。
  5. 将诊断结果复制给维护者进一步分析。
